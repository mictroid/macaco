committed-but-unshipped on top of vc66 — plus three fixes found during S8+ (`ce03171353d9547e0c`)
on-device testing:

- **Profile header avatar clip** (`ProfileScreen.kt`): the scrollable body was pulled up under the
  Scaffold `topBar` via `.offset(y = -bannerBottomPadding)`; since Compose draws the topBar over the
  body, the teal banner painted over the top of the avatar (worst on tablets). Removed the offset so
  the avatar sits flush below the banner. Verified on device.
- **Print Book 0-byte PDF** (`PrintBookExporter.kt`, `PrintExportScreen.kt`, `JournalViewModel.kt`):
  `decodeForPrint`'s bounds `openInputStream` was unguarded — a stale/unknown media URI (a cloud-synced
  entry whose photo was taken on another device) threw `FileNotFoundException`, which aborted the entire
  export and left the pre-created SAF file at 0 bytes. Fixes: (1) `decodeForPrint` fully `runCatching`-
  wrapped → one unreadable photo becomes a placeholder page, not a dead export; (2) bitmaps recycle
  only AFTER `finishPage` (recycle-before-finish also crashes the PdfDocument); (3) entry photos resolve
  through the Drive cache (`resolvePhotoUri` + new `BookConfig.cachedDrivePhotos`), like the rest of the
  app; (4) the VM awaits `ensurePhotosCached()` before rendering so Drive-only photos download first;
  (5) `PrintExportScreen` now surfaces the `Error` state (it previously only handled `Ready`, so failures
  were silent). Verified on device: a proper multi-page PDF now generates. (On the S8+ test account the
  photos live in a different account's Drive, so they render as placeholders — a test-data artifact.)
- **Camera-roll suggestion permission** (`JournalListScreen.kt`): the gate required `READ_MEDIA_IMAGES
  && ACCESS_MEDIA_LOCATION`, but Android 14+ won't grant ACCESS_MEDIA_LOCATION in the same request as the
  read permission, so tapping "Allow" left the gate false and nothing happened. Now gated on the read
  permission alone, with ACCESS_MEDIA_LOCATION requested as a second step (best-effort GPS enrichment).

Release notes rewritten for the whole batch (On This Day widget, Print Book PDF, Year in Travel, camera-
roll suggestions, shared trip links, entry search + weather stamp, plus the fixes).
