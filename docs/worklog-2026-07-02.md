### Briefs

- **map-arrows-v14** (`MapScreen.kt`) ‚Äî reactive off-screen pin tracker (new `LaunchedEffect` keyed
  on `cameraPositionState.position`) replacing the one-time computation, fixing stale-coordinate
  longitude wraparound; nearest-pin navigation; `arrowVerticalFraction()` helper for dynamic vertical
  arrow placement toward the pin's latitude; navigate to the pin's actual lat; arrows decoupled from
  `globeSpanning` (now show whenever off-screen pins exist ‚Üí tablet landscape). Chevrons use theme
  tokens.
  - **Deviations:** brief claimed `density` was "already in scope" ‚Äî it was a local `val` inside the
    first `LaunchedEffect` only; hoisted `density` to composable scope (removed the inner decl).
    Added imports `IntOffset`, `kotlin.math.roundToInt`, `kotlin.math.PI` (all absent; brief's helper
    used bare `PI`). Now-dead `mapLatCenter` reads left in place (harmless; still written).

- **ui-polish** (`NewEditEntryScreen.kt`, `MapScreen.kt`) ‚Äî date picker branded via
  `DatePickerDefaults.colors()` (surface container + primary accents); landscape map header icon
  `size(24dp)` ‚Üí `size(20dp).offset(y=-2dp)`.
  - **Deviation:** added `import androidx.compose.material3.DatePickerDefaults` (absent).

- **adventure-reel-v3** (`AdventureReelEncoder.kt`) ‚Äî Ken Burns continuity across the dissolve
  boundary (cap `t` at 1.0; main loop starts at `FADE_FRAMES` for photos after the first); four
  pre-allocated class-level Paints replacing per-frame allocation (A53 GC-stall fix).
  - **Deviation:** the brief's literal main-loop snippet started ALL photos at `FADE_FRAMES`, which
    would skip the first photo's opening pan. Used `mainStart = if (prev != null) FADE_FRAMES else 0`
    to match the brief's own prose (first photo has no dissolve). Frame counts unchanged.

- **journal-landscape** (`JournalListScreen.kt`) ‚Äî landscape header brand content centred between
  fixed 40dp flanking elements; landscape-aware compact drawer header (slim Row) so bottom nav
  (sign-out) stays on-screen. Portrait unchanged.

- **profile-landscape-v2** (`ProfileScreen.kt`) ‚Äî 2-pane now activates on tablets
  (`|| screenWidthDp >= 600`); left-pane top spacer 16dp ‚Üí 8dp.

- **settings-landscape** (`SettingsScreen.kt`) ‚Äî compact landscape header (slim Row); Theme Color +
  Background Image side-by-side in landscape (weight(1f) each); picker button 80dp ‚Üí 44dp inline row
  in landscape (no-image state).
  - **Deviation:** brief claimed `LocalConfiguration` was already imported ‚Äî it was NOT (build failed
    on it); added `import androidx.compose.ui.platform.LocalConfiguration`. Extracted the two cards to
    `@Composable () -> Unit` lambda vals (rather than duplicating the large card bodies) so both
    orientations share one definition; hint text hidden in landscape to save vertical space.

- **language-switch-flash** (`res/values/themes.xml`, `MainActivity.kt`) ‚Äî added
  `android:windowBackground=@color/splash_background` to `Theme.Macaco.Splash` (was inheriting black);
  guarded `installSplashScreen()` with `if (savedInstanceState == null)` so recreations (locale/
  rotation) skip the theme-swap frame. Kills the first-language-switch black flash in dark mode.
